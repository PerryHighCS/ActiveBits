# Algorithm Demo Activity - Quick Start

## ğŸ¯ For Instructors

### Setup
1. Start a new session: Go to your teacher dashboard
2. Click "Algorithm Demonstrations"
3. Share the session ID with students

### Demo a Sorting Algorithm
1. Students join the session
2. Click algorithm picker, select "Selection Sort"
3. Use "Next Step" button to advance
4. Watch pseudocode highlight in real-time
5. All students see the same visualization
6. Click "Generate New Array" to restart with new data

### Demo Binary Search
1. Select "Binary Search"
2. Algorithm displays target number
3. Click "Next Step" to divide the search space
4. Notice eliminated elements fade out
5. Watch as algorithm converges on target

### Interactive Guessing Game
1. Select "Binary Search Guessing Game"
2. Enter guesses and click "Guess"
3. Get feedback (higher/lower)
4. Students watch and count how many guesses it takes
5. Great for showing binary search efficiency!

- Clear browser cache

- Click "Reset" to restart

## ğŸ“ Class Activities

### 1. Sorting Race
- Show selection sort (O(nÂ²))
- Show insertion sort (O(nÂ²))
- Count steps - discuss why similar complexity
- Show with nearly-sorted data

### 2. Search Showdown
- Binary search: ~7 steps for 1000 items
- Linear search: could be 1000 steps
- Students vote on which is faster before demo

### 3. Recursion Discovery
- Show factorial computation
- Watch call stack build up
- See how results combine coming back down
- Discuss base cases

### 4. Efficiency Challenge
- Students predict: how many guesses needed for binary search?
- Usually guess too high
- Demonstrate logarithmic efficiency
